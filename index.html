What is css?
  1. CSS stands for cascading stylesheet.
  2. CSS describes how html elements are to be displayed on screen, paper, or in other media
  3. Css is not necessary Css saves lot of work. It can control the layout of multiple web pages all at once
  4. External stylesheet are stored in CSS files

CSS can be added to html documents in 3 ways

  • Inline - By using the style attribute inside html elements.
  • Internal - By using the style element in head section.
  • External - By using the link element to link an external css file.
  • The most common to add css, is to keep the styles in external css files. However, in this tutorial we will use inline and internal styles, because this is easier to demonstrate, and easier to you to try it yourself.

when comparing CSS class vs ID, the difference is that css class applies style to multiple elements. ID, on the other hand, applies style to unique element. ID is also special in that you can us especial URL to link directly to an element and its used by JavaScript.

In css, selectors are used to target a specific element or range of elements on a web page. Once an element has been targated, a style or set of styles can be applied to the element.

There is wide range of selectors available. Two of the mainly used are class and ID. Both are used to target element to which styles should be applied.
